# Installation process

```bash
git clone github.com/vizvasrj/blog
```

```bash
cd blog
python3 -m venv ../zoo/blog
pip install -r requirements.txt
./manage.py runserver
```

http://127.0.0.1:8000/ will be address which you need to get connect,
